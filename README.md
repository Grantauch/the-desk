# the desk.

Classroom home base for US History, Economics, and Law — live at [grant-desk.com](https://grant-desk.com).

Built with [Astro](https://astro.build) + [Tailwind CSS](https://tailwindcss.com), deployed on Netlify.

- **New here?** Read `START-HERE.md`.
- **Making an edit?** Read `EDITING.md`.
- **AI maintenance conventions:** `CLAUDE.md`.

```sh
npm install    # once
npm run dev    # local site at localhost:4321
npm run build  # production build (must pass before pushing)
```
